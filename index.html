    📚 科学上网完全指南¶

    一本从零基础到精通的科学上网技术完整教程

    📖 关于本书¶

    《科学上网完全指南》是一本系统全面的技术教程，涵盖从基础理论到实战部署的完整知识体系。本书已更新至2025年最新版本，包含Reality、Hysteria 2等最新代理协议，以及现代化客户端工具的使用指南。旨在帮助读者深入理解网络代理技术原理，掌握各种工具的使用方法，并在合法合规的前提下实现高效、安全的网络访问。

    ✨ 核心特点¶

    • 🎯 系统全面 - 从零基础到高级应用的完整知识体系
    • 💻 实战导向 - 大量配置示例、命令脚本、实战案例
    • 📝 深入浅出 - 原理讲解 + 实践操作，循序渐进
    • 🔒 注重安全 - 专门章节讲解安全防护和隐私保护
    • ⚖️ 合规意识 - 强调法律法规和合理使用边界

    📊 内容规模¶

    • 总文件数：27个
    • 正文章节：22章
    • 附录资料：4个
    • 总字数：约 60-70万字

    🆕 2025年重要更新¶

    新增协议与技术¶

    • 🌟 Reality协议 - 无需域名和证书的TLS伪装技术
    • 🚀 Hysteria 2 - 基于QUIC的高性能代理协议
    • 🔐 Firefox 145反指纹技术 - 浏览器隐私保护最新进展
    • 🌐 DNS over QUIC (DoQ) - 新一代加密DNS标准
    • 📱 Clash Verge Rev - 替代已停止维护的Clash for Windows
    • 🎛️ 3X-UI面板 - 支持Reality协议的管理面板

    更新的客户端工具¶

    • Windows: Clash Verge Rev（替代CfW）
    • macOS: Clash Verge Rev（跨平台统一）
    • Android: Clash Meta、NekoBox（新推荐）
    • Linux: Clash Meta（功能增强）

    隐私保护新内容¶

    • 浏览器指纹防护（Firefox 145、Brave、Tor）
    • WebRTC泄漏防护（Manifest V3兼容扩展）
    • 加密DNS最佳实践（DoH/DoT/DoQ）
    • 美国联邦政府加密DNS强制要求

    已废弃工具标注¶

    • ⚠️ Clash for Windows（2023.11停止维护）
    • ⚠️ Clash for Android原版（2023归档）
    • ⚠️ X-UI原版（建议迁移3X-UI）
    • ⚠️ Qv2ray、V2RayX、SagerNet（已停止维护）

    📚 章节概览¶

    第一部分：基础知识篇¶

    介绍科学上网的基本概念、网络基础知识和核心技术原理，为后续学习打下坚实基础。

    第二部分：主流协议与工具篇¶

    详细讲解 Shadowsocks、V2Ray、Xray、Reality、Hysteria 2、Trojan、Clash 等主流代理协议，以及各平台客户端工具的使用方法。重点介绍Reality协议和Hysteria 2高性能方案。

    第三部分：高级配置篇¶

    深入探讨订阅管理、分流规则、性能优化等高级话题，提升使用效率。

    第四部分：服务器搭建篇¶

    从 VPS 选购到服务器部署，手把手教你搭建自己的代理服务器。包含Reality协议一键部署、3X-UI面板管理、Hysteria 2配置等最新内容。

    第五部分：安全与隐私篇¶

    重点关注网络安全防护、隐私保护、日志管理等安全话题。新增浏览器指纹防护、WebRTC泄漏防护、Firefox 145反追踪技术等最新隐私保护内容。

    第六部分：故障排查篇¶

    系统化的故障诊断方法和网络测试工具使用指南。

    第七部分：最佳实践篇¶

    针对不同应用场景的最佳实践方案，以及成本优化和合规使用建议。
