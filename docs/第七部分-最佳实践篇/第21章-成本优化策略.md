# 第21章 成本优化策略

科学上网需要持续投入。本章介绍如何在保证质量的前提下优化成本。

## 21.1 机场服务对比

### 21.1.1 机场分类

**按价格分类：**

```
入门级（月付10-30元）：
特点：
- 基础线路
- 流量限制较多
- 节点数量少
- 稳定性一般
适合：偶尔使用、预算有限

中端级（月付30-80元）：
特点：
- CN2/BGP线路
- 流量充足（100GB-500GB）
- 节点较多
- 稳定性较好
适合：日常使用、轻度办公

高端级（月付80-200元）：
特点：
- IPLC/IEPL专线
- 大流量/不限流
- 节点众多
- 稳定性极高
- 流媒体解锁
适合：重度使用、专业需求

顶级（月付200元以上）：
特点：
- 全专线
- 不限流量
- 全球节点
- 7×24客服
- 企业级稳定性
适合：商业用途、团队使用
```

### 21.1.2 机场评估标准

**关键指标：**

```
1. 线路质量
   ✓ CN2 GIA优于CN2 GT
   ✓ IPLC/IEPL最优
   ✓ 普通线路次之
   
   测试方法：
   - 查看线路说明
   - 测试延迟和速度
   - 高峰期测试

2. 流量配额
   评估自身需求：
   - 日常浏览：50-100GB/月
   - 视频观看：200-500GB/月
   - 下载需求：500GB+/月
   
   计算方法：
   1080p视频约3GB/小时
   4K视频约7GB/小时

3. 节点数量与分布
   ✓ 香港/台湾/日本/新加坡
   ✓ 美国西海岸/东海岸
   ✓ 欧洲主要国家
   
   避免：
   ❌ 仅单一地区
   ❌ 冷门地区为主

4. 稳定性
   考察：
   - 运营时长（>1年）
   - 用户评价
   - 售后服务
   - 公告更新频率
   
   警告信号：
   ❌ 频繁跑路传闻
   ❌ 长期无更新
   ❌ 客服失联

5. 附加功能
   加分项：
   ✓ 流媒体解锁
   ✓ 游戏加速
   ✓ ChatGPT支持
   ✓ 多设备支持
   ✓ 客户端支持完善

6. 性价比
   计算公式：
   性价比 = (流量GB × 线路质量系数) / 价格
   
   线路质量系数：
   - 普通线路：1.0
   - CN2 GT：1.5
   - CN2 GIA：2.0
   - IPLC：3.0
```

### 21.1.4 避坑指南

**常见陷阱：**

```
1. 超低价陷阱
   月付5-10元，宣称不限速
   
   真相：
   - 超售严重
   - 高峰期不可用
   - 随时跑路风险
   
   识别：
   价格远低于成本（IPLC成本至少50元/月）

2. 虚假宣传
   宣传：
   "100%解锁Netflix"
   "1Gbps带宽"
   "永久免费"
   
   现实：
   - 部分节点解锁
   - 实际速度远低
   - 免费节点不稳定
   
   验证：
   试用后再购买

3. 跑路风险
   特征：
   - 新机场大力促销
   - 年付优惠力度大
   - 无退款政策
   
   防范：
   - 月付为主
   - 避免长期套餐
   - 分散购买

4. 过度超售
   表现：
   - 晚高峰速度骤降
   - 频繁断线
   - 节点经常满载
   
   识别：
   - 试用高峰期
   - 查看用户评价
   - 对比不同时段

5. 隐私风险
   问题机场：
   - 记录用户访问
   - 出售用户数据
   - 被执法部门控制
   
   防范：
   - 选择知名机场
   - 避免使用真实信息
   - 加密敏感通信
```

## 21.3 流量节省技巧

### 21.3.1 智能分流

**精细化规则：**

```yaml
# Clash优化配置
rules:
  # 国内网站直连
  - DOMAIN-SUFFIX,baidu.com,DIRECT
  - DOMAIN-SUFFIX,taobao.com,DIRECT
  - DOMAIN-SUFFIX,qq.com,DIRECT
  - DOMAIN-SUFFIX,163.com,DIRECT
  - DOMAIN-SUFFIX,sina.com.cn,DIRECT
  
  # 国内CDN直连
  - DOMAIN-SUFFIX,cdn.jsdelivr.net,DIRECT
  - DOMAIN-SUFFIX,unpkg.com,DIRECT
  
  # 大文件下载直连（如果国内有镜像）
  - DOMAIN-SUFFIX,mirrors.tuna.tsinghua.edu.cn,DIRECT
  - DOMAIN-SUFFIX,mirrors.aliyun.com,DIRECT
  
  # 需要代理的才走代理
  - DOMAIN-SUFFIX,google.com,PROXY
  - DOMAIN-SUFFIX,youtube.com,PROXY
  - DOMAIN-SUFFIX,github.com,PROXY
  
  # GeoIP CN直连
  - GEOIP,CN,DIRECT
  
  # 其他走代理
  - MATCH,PROXY

效果：
节省50%以上流量
避免国内访问走代理
```

**流量统计分析：**

```bash
#!/bin/bash
# traffic_analysis.sh

# 获取Clash流量统计
curl -s http://127.0.0.1:9090/traffic | jq

# 计算今日使用
TODAY_UP=$(vnstat --oneline | cut -d';' -f4)
TODAY_DOWN=$(vnstat --oneline | cut -d';' -f5)

echo "今日上传：$TODAY_UP"
echo "今日下载：$TODAY_DOWN"

# 预警
LIMIT_GB=100
CURRENT_GB=$(vnstat --oneline | cut -d';' -f11)

if [ $CURRENT_GB -gt $LIMIT_GB ]; then
    echo "警告：月流量已超过${LIMIT_GB}GB"
fi
```

### 21.3.4 减少浪费

**常见浪费：**

```
1. 自动更新走代理
   Windows更新
   App Store更新
   软件自动更新
   
   解决：
   - 设置更新时段
   - 手动更新
   - 更新时关闭代理

2. 云同步走代理
   OneDrive同步
   iCloud同步
   Dropbox同步
   
   解决：
   - 暂停自动同步
   - 选择性同步
   - 同步时关闭代理

3. 后台应用
   自动下载
   推送通知
   数据同步
   
   解决：
   - 关闭不必要的后台
   - 限制应用联网
   - 定期检查流量

4. 视频自动播放
   YouTube自动播放
   Twitter视频
   Facebook视频
   
   解决：
   - 禁用自动播放
   - 降低视频质量
   - WiFi环境观看
```
